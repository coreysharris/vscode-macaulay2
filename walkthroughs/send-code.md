Use `Shift+Enter` or `Ctrl+Enter` to send selected code to the webview REPL.

Use the run button in the editor title area to send the whole `.m2` file to the webview REPL.

The terminal send command is available from commands and menus without a default keybinding. Note also that this extension comes with a formatter -- you can use a shortcut to automatically format macaulay2 files.

Use **Open Macaulay2 Keyboard Shortcuts** to inspect or rebind the extension shortcuts.
